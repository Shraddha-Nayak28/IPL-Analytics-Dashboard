Dashboard coverage

4 Report Pages in Power BI

Overview — Season-wise match count, total runs, title winners leaderboard
Team Performance — Win rates, toss decision impact, head-to-head stats
Batting Analysis — Top run scorers, powerplay vs death over trends
Venues & Bowling — Top wicket takers, venue-wise outcomes, city map
Key findings
Teams choosing to field first after winning the toss win ~52% of the time
Mumbai Indians leads all-time win count across IPL seasons
Death overs consistently produce higher run rates than powerplay overs
Wankhede & Chinnaswamy are the highest-scoring venues in IPL history

How to explore this project
1
Download matches.csv and deliveries.csv from the Kaggle link in README.md
2
Open ipl_analysis_queries.sql in DB Browser for SQLite (free) to explore the raw SQL analysis
3
Open IPL_Analytics_Dashboard.pbix in Power BI Desktop (free) and point data source to local CSV files
4
Use the slicers on each dashboard page to filter by season, team, and venue
About this project

I'm an engineering graduate exploring roles in data analytics and business intelligence. This project was built entirely on my own initiative using a publicly available Kaggle dataset — to practice a full BI workflow from raw data ingestion → SQL analysis → Power BI dashboard. Cricket data made it engaging to dig into, and the project helped me understand how real reporting tools connect structured data to visual insights.

Dataset: Kaggle · IPL Complete Dataset (2008–2024)  |  Built with Power BI Desktop & SQL  |  June 2026
